# 📷 標識画像の入れ替えガイド

このフォルダ（`img/`）に対応するファイル名の画像を置くだけで、
クイズの標識イラストが自動的にその画像に切り替わります。

画像ファイルが**存在しない場合は SVG イラストが代わりに表示**されます（フォールバック）。
開発中でも問題なく動作しますので、準備できた画像から順番に置いていただければ大丈夫です。

## 画像の仕様（推奨）

| 項目 | 推奨値 |
|---|---|
| ファイル形式 | PNG / JPG / WebP（拡張子は `.png` 推奨） |
| サイズ | 200×200px 以上（正方形または正方形に近い比率） |
| 背景 | 白または透明（PNG 推奨） |
| ファイルサイズ | 1枚あたり 200KB 以下を推奨 |

> **注意**：ファイル名は上表の通りに正確に合わせてください（大文字・小文字・アンダーバーを含めて一致させること）。

---

## 入れ替え手順

1. 上の一覧で置き換えたい標識のファイル名を確認する
2. 対応する画像ファイルを用意し、上記ファイル名にリネームする
3. このフォルダ（`img/`）に配置する
4. ブラウザを再読み込みすると自動的に新しい画像が表示される

---

## ファイル形式を変えたい場合（JPG / WebP など）

`js/quiz.js` の `QUIZ_DATA` 内、各問題の `signImage` プロパティのパスを変更してください。

```js
// 例: Q1 の画像を JPG に変更する場合
{
  id: 1,
  signImage: 'img/sign_01_no_entry.jpg',  // ← ここを変更
  ...
}
```

---

## カスタムパスを使いたい場合

サブフォルダやまったく別のファイル名も使えます。

```js
// 例: img/signs/ フォルダに整理したい場合
signImage: 'img/signs/tsuukoudome.png',
```

パスはすべて `index.html` からの相対パスで記述してください。
